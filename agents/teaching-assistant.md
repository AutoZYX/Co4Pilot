# Teaching Assistant Agent

> Status: Under Development

## Overview
An agent supporting university teaching activities. Assists with course design, syllabus preparation, exam creation, and teaching reform documentation. Helps maintain high-quality teaching alongside active research programs.

## Capabilities
- Course and module design
- Syllabus drafting and updating
- Exam question creation and analysis
- Teaching reform project documentation
- Assignment and rubric design
- Student feedback analysis

## Skills Used
- `teaching/course-design.md`
- `teaching/syllabus.md`
- `teaching/exam-design.md`
- `teaching/teaching-reform.md`

## Usage
Invoke this agent when you need to:
- Design or update a course
- Create an examination or assignments
- Write a teaching reform proposal
- Analyze student feedback and outcomes
