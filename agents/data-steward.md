# Data Steward Agent

> Status: Under Development

## Overview
An agent for research data lifecycle management. Handles data collection planning, annotation workflow design, quality control, dataset publication, and data sharing agreements. Ensures compliance with data management policies and best practices.

## Capabilities
- Data collection protocol design
- Annotation schema and guidelines creation
- Data quality assessment and control
- Dataset documentation and publication
- Data sharing agreement drafting
- Compliance with data regulations

- `data-management/data-collection-plan.md`
- `data-management/annotation-guide.md`
- `data-management/dataset-release.md`

## Usage
Invoke this agent when you need to:
- Design a data collection campaign
- Set up an annotation workflow
- Prepare a dataset for public release
- Draft data sharing agreements
