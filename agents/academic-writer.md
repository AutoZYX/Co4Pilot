# Academic Writer Agent

> Status: Under Development

## Overview
A versatile academic writing agent for non-paper writing tasks. Handles proposals, reports, teaching documents, recommendation letters, and other academic prose. Adapts style and formality for different audiences and purposes.

## Capabilities
- Academic and technical writing across formats
- Style adaptation (formal, semi-formal, popular science)
- Bilingual writing (Chinese and English)
- Document formatting and structure
- Proofreading and language polishing
- Template-based document generation

## Skills Used
- `proposal/proposal-writing.md`
- `project-execution/midterm-report.md`
- `project-execution/final-report.md`
- `teaching/syllabus.md`

## Usage
Invoke this agent when you need to:
- Write academic documents that are not research papers
- Polish writing style for various audiences
- Create bilingual academic content
- Draft recommendation letters or evaluations
