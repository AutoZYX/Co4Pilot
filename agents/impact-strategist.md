# Impact Strategist Agent

> Status: Under Development

## Overview
An agent for maximizing academic impact and visibility. Manages research dissemination across platforms, tracks citation metrics, prepares talent program applications, and maintains professional profiles. Supports both Chinese and international academic ecosystems.

## Capabilities
- Research dissemination strategy
- WeChat and LinkedIn content creation
- Invited talk and presentation preparation
- Media interview preparation
- CV and profile maintenance
- Citation tracking and analysis
- Talent program application support

## Usage
Invoke this agent when you need to:
- Plan a dissemination strategy for a new publication
- Prepare for an invited talk or media interview
- Update your CV and track academic metrics
- Apply for a talent program or award
