# Project Manager Agent

> Status: Under Development

## Overview
An agent for managing funded research projects from kickoff to closeout. Tracks milestones, manages budgets, prepares progress reports, and ensures compliance with funder requirements. Handles multiple concurrent projects.

## Capabilities
- Project milestone planning and tracking
- Budget monitoring and forecasting
- Midterm and final report drafting
- Audit preparation and compliance checks
- Multi-project dashboard management
- Deliverable tracking (papers, patents, students)

## Skills Used
- `project-execution/midterm-report.md`
- `project-execution/final-report.md`
- `project-execution/budget-management.md`
- `project-execution/audit-prep.md`

## Usage
Invoke this agent when you need to:
- Track progress on funded research projects
- Prepare midterm or final project reports
- Monitor budget execution across projects
- Prepare for a project audit
