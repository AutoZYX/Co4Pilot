# Invention Disclosure

> Status: Under Development

## Overview
Draft invention disclosure documents for university technology transfer offices. Captures technical details, prior art context, potential applications, and inventor contributions in standardized formats.

## Usage
Use this skill when you need to:
- Write an invention disclosure for the TTO
- Document the technical innovation clearly
- Describe potential commercial applications
- List inventor contributions and prior art
