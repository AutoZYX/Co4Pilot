# Proposal Defense Preparation

> Status: Under Development

## Overview
Prepare for proposal defense presentations (e.g., NSFC panel review, thesis proposal defense). Covers slide design, anticipating questions, rehearsal strategies, and response preparation.

## Usage
Use this skill when you need to:
- Prepare slides for a proposal defense
- Anticipate and prepare for reviewer questions
- Structure a concise and compelling presentation
- Develop backup slides for technical details
