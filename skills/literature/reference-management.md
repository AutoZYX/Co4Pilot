# Reference Management

> Status: Under Development

## Overview
Organize, format, and manage bibliographic references. Supports BibTeX, EndNote, and various citation styles (APA, IEEE, GB/T 7714). Handles deduplication, metadata correction, and citation key generation.

## Usage
Use this skill when you need to:
- Convert references between citation formats
- Clean and deduplicate a bibliography
- Generate properly formatted reference lists
- Check citation completeness and consistency
