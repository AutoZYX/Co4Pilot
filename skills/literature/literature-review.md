# Literature Review

> Status: Under Development

## Overview
Synthesize and organize literature into structured reviews. Supports thematic analysis, chronological surveys, gap identification, and comparative tables. Outputs structured summaries suitable for paper introduction sections or standalone review articles.

## Usage
Use this skill when you need to:
- Write a literature review section for a research paper
- Identify research gaps and trends
- Create comparative analysis tables
- Generate annotated bibliographies
