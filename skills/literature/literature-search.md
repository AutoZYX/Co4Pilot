# Literature Search

> Status: Under Development

## Overview
Systematic literature search skill for identifying relevant publications across databases (Google Scholar, Semantic Scholar, PubMed, IEEE Xplore, CNKI, etc.). Supports structured search strategies, keyword expansion, and snowball sampling.

## Usage
Use this skill when you need to:
- Conduct a comprehensive literature search on a research topic
- Build a search strategy with Boolean operators
- Identify seminal papers and recent advances
- Search across Chinese and English databases
