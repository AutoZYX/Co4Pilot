# Editor Communication

> Status: Under Development

## Overview
Draft professional correspondence with journal and conference editors. Handles submission inquiries, status follow-ups, appeals, and editorial decision responses.

## Usage
Use this skill when you need to:
- Write to an editor about a submission status
- Draft an appeal for a rejected manuscript
- Respond to editorial decision letters
- Communicate about special issue proposals
