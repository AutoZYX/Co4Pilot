# Data Agreement

> Status: Under Development

## Overview
Draft and review data sharing agreements, data usage licenses, and collaboration data terms. Covers DUA (Data Use Agreement), NDA considerations, GDPR compliance, and Chinese data protection requirements.

## Usage
Use this skill when you need to:
- Draft a data sharing agreement
- Review data usage terms from partners
- Ensure compliance with data protection regulations
- Set up data collaboration frameworks
