# Dataset Release

> Status: Under Development

## Overview
Prepare datasets for public release and publication. Covers data documentation (datasheets, data cards), license selection, anonymization, format standardization, and repository hosting (Zenodo, Hugging Face, etc.).

## Usage
Use this skill when you need to:
- Prepare a dataset for public release
- Write datasheets or data cards
- Choose appropriate data licenses
- Anonymize and format data for publication
