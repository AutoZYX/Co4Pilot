# Final Report

> Status: Under Development

## Overview
Draft final reports for completed research projects. Comprehensive documentation of objectives achieved, methodology, results, publications, patents, trained students, and impact assessment. Supports various funder-specific formats.

## Usage
Use this skill when you need to:
- Write a comprehensive final project report
- Document all deliverables and outcomes
- Summarize publications, patents, and student training
- Assess impact and future research directions
