# Midterm Report

> Status: Under Development

## Overview
Draft midterm progress reports for funded research projects. Covers research progress summary, milestone achievement, preliminary results, budget execution, and next-phase plans. Supports NSFC, MOST, and provincial fund reporting formats.

## Usage
Use this skill when you need to:
- Write a midterm progress report for a funded project
- Summarize research achievements against planned milestones
- Report budget execution status
- Outline adjusted plans for remaining project period
