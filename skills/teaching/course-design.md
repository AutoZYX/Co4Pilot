# Course Design

> Status: Under Development

## Overview
Design university courses from scratch or redesign existing ones. Covers learning objectives (Bloom's taxonomy), module planning, assessment design, active learning integration, and course material preparation.

## Usage
Use this skill when you need to:
- Design a new course from scratch
- Redesign an existing course with new objectives
- Plan course modules and weekly schedules
- Design blended or flipped classroom approaches
