# Syllabus Writing

> Status: Under Development

## Overview
Draft comprehensive course syllabi meeting university requirements. Covers course description, learning outcomes, weekly schedule, grading policy, textbook selection, academic integrity policy, and accessibility statements.

## Usage
Use this skill when you need to:
- Write a complete course syllabus
- Update an existing syllabus for a new semester
- Align syllabus with university template requirements
- Design grading rubrics and assessment policies
