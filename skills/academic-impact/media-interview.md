# Media Interview Preparation

> Status: Under Development

## Overview
Prepare for media interviews about research findings. Covers key message development, sound bite crafting, technical-to-lay translation, and crisis communication. Supports both Chinese and English media outlets.

## Usage
Use this skill when you need to:
- Prepare talking points for a media interview
- Translate technical findings into accessible language
- Develop key messages and sound bites
- Handle sensitive or controversial research topics
