# Cover Letter

> Status: Under Development

## Overview
Draft cover letters for journal and conference submissions. Highlights novelty, significance, and fit with the target venue. Supports suggesting reviewers and addressing editor concerns.

## Usage
Use this skill when you need to:
- Write a cover letter for journal submission
- Highlight paper novelty and significance
- Suggest or exclude potential reviewers
- Address specific journal requirements
