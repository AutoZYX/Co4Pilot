# Conference Paper Writing

> Status: Under Development

## Overview
Write papers formatted for specific conference submissions. Handles page limits, LaTeX templates (IEEE, ACM, Springer LNCS), camera-ready preparation, and supplementary materials.

## Usage
Use this skill when you need to:
- Draft a paper for a specific conference (e.g., CVPR, NeurIPS, AAAI, IV, ITSC)
- Work within page limit constraints
- Prepare camera-ready versions
- Create supplementary material documents
